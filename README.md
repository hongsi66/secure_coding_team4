# secure_coding_team4
📸 AI 기반 인스타그램 클론 코딩 프로젝트
📌 프로젝트 개요

이 프로젝트는 AI를 활용한 인스타그램 클론 코딩을 목표로 합니다.
비개발자 관점에서 AI에게 기능 구현을 요청하여 SNS 형태의 웹 서비스를 완성하는 것을 목적으로 합니다.

🧩 주요 기능

🔐 자체 로그인/회원가입 기능 (구글 등 외부 로그인 금지)

🖼️ 사진 및 글 업로드 기능

👀 모든 사용자가 게시글 열람 가능 (팔로우 여부 무관)

🛠️ 기술 스택

프레임워크: 자유 선택 (예: Flask, Django, Node.js 등)

데이터베이스: SQLite3

프로그래밍 언어: 자유 (Python 권장)

🤖 개발 방식

AI(예: ChatGPT, Gemini 등)에게 기능 단위로 코드를 요청하여 개발

보안 관련 코드(SQL 인젝션 등)는 직접 요청 금지, 오직 기능 구현 중심으로 진행
